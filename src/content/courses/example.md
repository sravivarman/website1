## Course Overview

This course introduces students to the fundamental concepts of computer programming using Python. Students will learn problem-solving techniques, algorithm design, and implementation skills essential for further study in computer science.

**Course Type:** Theory + Lab  
**Contact Hours:** 3 Lectures + 2 Lab hours per week  
**Prerequisites:** None

## Syllabus

### Unit 1: Introduction to Programming (8 hours)
- Introduction to computers and programming
- Programming languages and paradigms
- Python installation and environment setup
- Variables, data types, and operators
- Input/output operations

### Unit 2: Control Structures (10 hours)
- Conditional statements (if, elif, else)
- Loops (for, while)
- Loop control statements (break, continue, pass)
- Nested control structures
- Problem-solving with control structures

### Unit 3: Functions (8 hours)
- Introduction to functions
- Function definition and calling
- Parameters and arguments
- Return values
- Scope and lifetime of variables
- Recursive functions

### Unit 4: Data Structures (10 hours)
- Lists and list operations
- Tuples
- Dictionaries
- Sets
- String manipulation

### Unit 5: File Handling & Modules (9 hours)
- File operations (read, write, append)
- Working with CSV files
- Introduction to modules
- Creating and importing modules
- Standard library overview
